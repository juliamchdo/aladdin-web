# Instalação 

### 1) Clone do repositório
```bash
git clone https://github.com/juliamchdo/aladdin-web.git
```

### 2) Acesse a pasta do projeto
```bash
cd aladdin-web
```

### 3) Instale as dependências
```bash
npm install (ou npm i)
```

### 4) Inicie o projeto
```bash
npm run dev
```
